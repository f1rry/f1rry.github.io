    • markdown的简单使用
    发表于 2018-07-06| 更新于 2020-01-04
    字数总计:924|阅读时长: 3 分钟
    | 阅读量:

    Markdown 是一种轻量级的「标记语言」，优点在于

    • 专注你的文字内容而不是排版样式。
    • 轻松的导出 HTML、PDF 和本身的 .md 文件。
    • 纯文本内容，兼容所有的文本编辑器与字处理软件。
    • 可读，直观。适合所有人的写作语言

    一、标题

    # ~ ######，#号的个数表示几级标题，即表示一级标题到六级标题

    二、强调

    1.*文字*         斜体字
2.**文字**       粗体字
3.***文字***     斜粗体字
4.用"_"代替"*"拥有同样效果

    以下为实例：

    1. 文字
    2. 文字
    3. 文字

    三、分段

    使用两个回车
例如：你好  这是一个分段
      你好这里没有分段

    以下为实例：
    你好 这是一个分段
    你好这里没有分段

    四、换行

    使用两个空格 + 回车
例如：唔慕，  /*这里有两个空格 + 回车*/
      这里有换行哦

    以下为实例：
    唔慕，
    这里有换行哦

    五、引用

    基本使用：

    使用 > + 空格  开始引用
使用两个回车   结束引用
例如：
  > 这里有一个引用
  > 引用换行  
  > 结束引用

  结束引用

    以下为实例：

    这里有一个引用
    引用换行
    结束引用

    结束引用

    简略使用：

    以第一行使用 > + 空格 为开始，以两个回车为结束
例如：
  > 这里有一个引用
  引用换行  
  结束引用

  结束引用

    以下为实例：

    这里有一个引用
    引用换行
    结束引用

    结束引用

    嵌套使用：

    > 最外层引用
> > 多一个 > 嵌套一层引用
> > > 可以嵌套很多层

    六、列表

    无序列表：

    * ， + ， - 选其中之一，注意后面紧跟个空格
例如：
* 八重樱
* 傻子那
* 德莉傻

    以下为实例：

    • 八重樱
    • 傻子娜
    • 德莉傻

    有序列表：

    a用数字+.再紧跟一个空格（只要是数字就可以，自动排序）
例如：
    1. 八重樱
    3. 德莉傻
    2. 傻子娜

    以下为实例：

    1. 八重樱
    2. 德莉傻
    3. 傻子娜

    嵌套列表：

    例如：
* 八重樱  
  1. 兽耳  
    * 想摸
  2. 有个妹妹八重凛
* 德莉傻
  1. 傻
  2. 合法萝莉
    * 还是傻
  3. 学院长
* 傻子娜
  1. 傻
  2. 想喝芽衣妹纸
  3. 变身后女王范？？？

    以下为实例：

    • 八重樱
      1. 兽耳
        • 想摸
      2. 有个妹妹八重凛
    • 德莉傻
      1. 傻
      2. 合法萝莉
        • 还是傻
      3. 学院长
    • 傻子娜
      1. 傻
      2. 想喝芽衣妹纸
      3. 变身后女王范？？？

    七、表格

    运用 - 和 | 来制表格
例如：
列1   | 列2 | 列3 
----- | --- | ---- 
第1行 | 12  | 13  
第2行 | 22  | 23  
第3行 | 32  | 33

    八、代码块

    /*以四个空格为开头或使用
    Code

    以下为实例：

    c

    九、链接

    常用格式：
    1.文字链接：[链接名称](链接地址)
    2.网址链接：<链接地址>
例如：
这是f1rry的[github](https://github.com/f1rry)
<https://github.com/f1rry>

    以下为实例：
    这是f1rry的github
    https://github.com/f1rry

    高级链接技巧
这个链接用 1 作为网址变量[github][1]
然后在文档的结尾为变量赋值（网址）
  [1]: https://github.com/f1rry
例如:
这指向f1rry的[github][1]

    以下为实例：
    这指向f1rry的github
